# HECTA (Hand and Eye Coordination Training Application)

## Description
HECTA is a game that aims to improve the hand and eye coordination of the player. The game is a 2D wac-a-mole style game where the player has to click on the mole that appears on the screen.

This repository contains the backend of the application. The backend is a REST API that is used to store the scores of the players and to retrieve the scores of the players.

## Technologies
* Python3.9

## Installation

* Clone the repository
```bash
git clone git@github.com:DerekWSmith/HECTA-backend.git
```
* Create a virtual environment
```bash
python3 -m venv venv
```
* Activate the virtual environment
```bash
source venv/bin/activate
```
* Install the dependencies
```bash
pip install -r requirements.txt
```
* Start the server
```bash
python3 manage.py runserver
```
